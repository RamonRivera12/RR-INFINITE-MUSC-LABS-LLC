# RR Infinite Music Labs LLC Website

Official website for RR Infinite Music Labs LLC.

## How to use

1. Upload these files to your GitHub repository.
2. Replace `logo.png` with your original logo file and keep the filename as `logo.png`.
3. Enable GitHub Pages in repository Settings > Pages.
4. Select branch: `main`, folder: `/root`.
5. Save and open your live website link.
